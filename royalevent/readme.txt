RoyalEvent - wordpress тема.
Версия CMS - WordPress 4.9.7.

Инструкция по установке темы:

1. Скопировать директорию "royalevent/" из ветки master в "wp-content/themes/".
2. Из ветки package загрузить плагины в админ. панель: Plugins - Add New - Upload Plugin.
3. Из ветки package скопировать директорию "uploads/" в "wp-content/".
4. В базе данных заменить "your_website_url_here" на url своего сайта без последнего слэша.
5. Сделать импорт базы данных.
